# LevelUP

A two sided video streaming marketplace platform, featuring credit card payment capabilities, user role management, complex user interfaces and advanced database relationships. 

## Built with:

* [Rails 4](https://guides.rubyonrails.org/v4.0/) - The web framework used

* PostgreSQL - Database

* [Bootstrap](https://getbootstrap.com/docs/3.3/) - CSS framework


## Technologies Used

* Stripe API

* AWS

* jQuery
